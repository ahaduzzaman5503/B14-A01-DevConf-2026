# AI Challenge Prompt -- Conference Tracks Section

## Prompt

I have already completed a DevConf 2026 landing page using HTML and CSS.
The page already includes a Navbar, Hero Section, Meet the Speakers
section, Pricing section, and Footer. I need to create only one
additional AI-generated section called **"Conference Tracks"** that fits
seamlessly with the existing design.

Generate semantic HTML and clean CSS for a responsive section with the
following requirements:

-   Dark modern developer-conference theme.
-   Centered heading: **"Explore Conference Tracks"**.
-   Short subtitle below the heading.
-   Display **6 track cards** in a responsive grid:
    -   3 columns on desktop
    -   2 columns on tablet
    -   1 column on mobile
-   Each card should include:
    -   A technology icon (use Font Awesome icons).
    -   Track title:
        -   AI & Machine Learning
        -   Cloud & DevOps
        -   Frontend Engineering
        -   Cyber Security
        -   Mobile Development
        -   Startup & Career
    -   A short 2--3 line description.
    -   A small badge showing the number of sessions (e.g., "12
        Sessions").
-   Add smooth hover effects including:
    -   Lift animation
    -   Soft shadow
    -   Subtle icon scaling
-   Use semantic HTML (`section`, `article`, `h2`, `p`) and clean,
    well-organized CSS.
-   Keep spacing, typography, border radius, and card styles consistent
    with a professional conference landing page.
-   Do **not** include JavaScript.
-   Return **only the HTML and CSS code**.
